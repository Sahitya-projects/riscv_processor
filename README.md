# RISC-V Processor Design in Verilog

This project implements core modules of a simple RISC-V processor using Verilog.

## Modules Implemented

### ALU (Arithmetic Logic Unit)

Operations supported:
- ADD
- SUB
- AND
- OR

## Simulation

Simulation was performed using:
- Icarus Verilog
- GTKWave

## Test Case

Inputs:
a = 10  
b = 5  

Results:

| Operation | Result |
|----------|--------|
| ADD | 15 |
| SUB | 5 |
| AND | 0 |
| OR | 15 |

Waveform generated using GTKWave.
